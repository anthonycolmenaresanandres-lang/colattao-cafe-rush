# Demo Test Checklist

Use this checklist on the deployed URL before in-store demo usage.

- [ ] Open deployed URL on phone.
- [ ] Confirm app loads and start screen appears.
- [ ] Start game.
- [ ] Win game by reaching score 100.
- [ ] Confirm VisualFlashPass appears.
- [ ] Tap Claim Discount and confirm success state.
- [ ] Win again and confirm cooldown blocks repeat claim.
- [ ] Tap Share My Win and confirm:
- [ ] If Web Share is available, native share opens.
- [ ] If Web Share is unavailable, clipboard fallback message appears.
- [ ] Tap Play Again and confirm Phaser restarts.
- [ ] Confirm barista can clearly view pass/share proof on screen.
