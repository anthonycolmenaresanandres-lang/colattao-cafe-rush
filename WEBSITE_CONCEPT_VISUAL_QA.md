# /website-concept — Visual QA Punch-List

A focused pass/fail checklist for evaluating whether the `/website-concept`
page feels premium, readable, mobile-first, and emotionally persuasive before
showing the owner.

How to use: open the live route on a real phone (portrait) **and** desktop.
Mark each item `[ ]` (to check), `[x]` (passes), or `[!]` (needs fix). Capture
any failure in **Section 8 — Notes and fixes**.

Route: `https://colattao-cafe-rush.vercel.app/website-concept`

---

## 1. First impression

- [ ] Does it feel premium within 3 seconds of load?
- [ ] Does it unmistakably feel like **Colattao** (not a generic café template)?
- [ ] Does the hero make someone want to visit the café today?
- [ ] Is the logo crisp and well-placed over the hero banner?
- [ ] Does the espresso / ceramic-blue / gold palette read immediately?

## 2. Mobile crop quality (portrait phone)

- [ ] **Hero image** — interior banner crops well; key subject not cut awkwardly.
- [ ] **Product cards** (future homepage concept) — images fill cleanly, no stretching.
- [ ] **Cultural still-life** — mochila / sombrero / cup remain recognizable after crop.
- [ ] **Digital ecosystem** — phones, QR sign, stickers, latte still legible at small size.
- [ ] **Footer image grid** — barista + community images crop cleanly side by side.
- [ ] "What to order first" 4-card row reflows to 1–2 columns gracefully.
- [ ] No image overflow, letterboxing, or distortion on narrow screens.

## 3. Readability

- [ ] Body text stays legible over the page-wide background texture (overlay strong enough).
- [ ] Headings have clear contrast against espresso panels.
- [ ] Button contrast passes a glance test (gold on espresso, etc.).
- [ ] Section hierarchy is obvious: H1 → section H2 → card H3 → body.
- [ ] Spacing/whitespace feels intentional (golden-ratio rhythm), not cramped.
- [ ] Glow dividers separate sections without feeling busy.
- [ ] Small text is never placed directly on busy imagery without overlay protection.

## 4. Emotional buying pull

- [ ] **Steam** card evokes warmth / the first sip.
- [ ] **Pastry** imagery triggers appetite appeal.
- [ ] **Colombian soul** comes through (culture, not cliché).
- [ ] **Community** section feels welcoming and real.
- [ ] **Origin story** conveys single-origin craft and roots.
- [ ] **Digital ecosystem** feels modern and differentiating.
- [ ] Overall: the page makes you *feel* something, not just read facts.

## 5. Conversion flow

- [ ] **View Live Menu** → `/menu` works from every placement (hero, order-first, conversion card).
- [ ] **Play Café Rush** → `/` works from hero and conversion card.
- [ ] **Open Owner Presentation** → `/owner-presentation` works from hero, conversion card, footer.
- [ ] Website-concept internal navigation/scroll feels smooth.
- [ ] CTAs are repeated at natural decision points without feeling spammy.
- [ ] Every button has a clear, single, obvious destination.

## 6. Brand discipline

- [ ] No competitor / chain names anywhere in visible copy.
- [ ] No fake scarcity ("only today", "act now", countdowns).
- [ ] No unsupported claims ("best coffee", "guaranteed", "#1").
- [ ] No visual clutter — each section has room to breathe.
- [ ] Colors stay consistent (espresso / parchment / gold / ceramic blue).
- [ ] Typography stays disciplined (serif display + clean sans only).
- [ ] Current website is framed as an evolution, never insulted.

## 7. Owner demo readiness

Pick one:

- [ ] **Ready to show** — premium, readable, persuasive; demo as-is.
- [ ] **Needs minor polish** — strong, with small tweaks (spacing, a crop, a line of copy).
- [ ] **Needs major polish** — structural or readability issues to resolve first.
