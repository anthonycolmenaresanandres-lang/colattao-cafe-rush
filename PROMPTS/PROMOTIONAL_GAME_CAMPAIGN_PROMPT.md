# Promotional Game Campaign Prompt

Use this prompt when planning or generating a promotional mini-game campaign for Colattao, Fina Calle, AMMA, or a future local-business client.

```text
Create a promotional game campaign that follows the Fina Calle Promotional Game Policy.

Core rule: Every game must entertain first, but sell quietly through repetition, reward, product visibility, and menu conversion.

Client/business:
[client name]

Campaign product(s):
[seasonal drink, plate, pastry, offer, or menu item]

Game concept:
[short game idea or existing game module]

Required customer actions:
[menu view, QR scan, Instagram follow, ask staff, visit, share, request update]

Asset sources:
[approved logo/product/menu/campaign asset paths]

Constraints:
- Entertain first.
- Keep the client brand dominant.
- Use real menu products and approved seasonal campaign assets.
- Repeat product visuals naturally during play.
- Include a reward/menu conversion loop.
- Keep CTAs soft and honest.
- Do not invent prices, discounts, availability, fake counts, fake saved items, auth, payments, database, tracking, or dashboards.
- Preserve existing gameplay geometry when this is only a campaign/art swap.

Deliver:
1. Product-first game loop
2. Seasonal campaign loop
3. Asset list and placement
4. Soft-selling mechanics
5. Reward/menu conversion CTA
6. What not to implement yet
```
