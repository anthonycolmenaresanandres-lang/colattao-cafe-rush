# Ecosystem Map

## 1. Live Routes
- `/`
- `/menu`
- `/owner-presentation`
- `/website-concept`
- `/owner-command-center`

## 2. Purpose of Each Route
- `/`: Main customer entrypoint with Colattao Cafe Rush gameplay and reward flow.
- `/menu`: Mobile-friendly digital menu with current offerings and owner update/feedback touchpoints.
- `/owner-presentation`: Spanish-first owner-facing page explaining value, operations, privacy, stickers, and next steps.
- `/website-concept`: Premium future-site concept showing how Colattao can evolve into a full digital brand ecosystem.
- `/owner-command-center`: Game-style owner command center concept (owner/demo-facing) for visual module planning and operations storytelling.

## 3. Navigation Flow
- Game route (`/`) links to:
  - `/menu`
  - `/owner-presentation`
  - `/website-concept`
- Owner presentation route (`/owner-presentation`) links to:
  - `/`
  - `/menu`
  - `/website-concept`
  - `/owner-command-center`
- Website concept route (`/website-concept`) links to:
  - `/`
  - `/menu`
  - `/owner-presentation`
  - `/owner-command-center`
- Menu route (`/menu`) links back to game and includes owner update / feedback flow.
- Owner command center route (`/owner-command-center`) is owner/demo-facing only and currently static/mock.
- Customer-facing routes (`/` and `/menu`) do not link to `/owner-command-center`.

## 4. Business Purpose
- Customer engagement: gameplay creates a stronger in-store interaction loop.
- Menu access: fast QR-driven menu browsing on mobile.
- Owner pitch: clear, Spanish-first presentation of value and operations.
- Website vision: premium forward-looking digital identity concept.
- Owner control room: a game-style command surface for planning updates and roadmap visibility.
- Feedback capture: simple owner/customer input flow.
- Monthly report/retainer sales support: provides a narrative frame for analytics and maintenance value.
- Future module roadmap: shows what is active now vs locked future modules.
- Dashboard-style demo value: illustrates how client dashboards can feel like premium game interfaces.
- Future white-label sales: reusable structure for additional local business clients.

## 5. White-Label Note
Future clients can reuse this same route structure with their own:
- theme
- assets
- menu
- copy
- QR signs

This keeps delivery consistent while allowing each brand to feel custom.

## 6. Owner Command Center Constraints
- Static/mock-only interface.
- No backend.
- No auth.
- No database.
- No payment processing.
- No real analytics fetching.
